# VB.NET-Link-Opener
A simple VB.NET application that opens websites in your default browser. Automatically adds https:// if missing.


A simple Windows application built with **VB.NET 2013** that opens websites in your default browser.

## Features

- 🔗 Enter any website URL and open it instantly
- 🔄 Automatically adds `https:// or .com ' if you forget it
- ⚠️ Shows error messages for invalid or empty URLs
- ⌨️ Press Enter key to open link (no need to click)
- 🎨 Clean, modern interface

## How to Run

1. Clone or download this repository
2. Open `link.sln` in Visual Studio 2013 or later

## How to Use

1. Type a website URL (e.g., `www.google.com` or `https://github.com`)
2. Click the **Open Website** button
3. Your default browser will open the website

## Technologies Used
- VB.NET
- Windows Forms
- .NET Framework
